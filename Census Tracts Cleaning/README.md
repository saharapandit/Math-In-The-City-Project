# Census Tracts Cleaning

This folder contains scripts and data for cleaning and visualizing census tract boundaries.

## Contents
- census_tract_cleaning.py: cleans tract data
- plot_tract_boundaries.py: plots tract boundaries
- Data/: input and cleaned output files
- notes.txt: working notes

## Setup
1. Create and activate a Python virtual environment.
2. Install required packages (for example: geopandas, pandas, matplotlib).

## Update Workflow
1. Make changes.
2. Commit.
3. Push.
